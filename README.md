# Linux Monitoring Lab 🚀

## 📌 Objective
To monitor system performance and understand resource utilization using basic Linux commands.

## 🛠️ Tools Used
- Linux (Kali/Ubuntu)
- Basic Shell Commands

## 🔍 Commands Practiced

### CPU Usage
top

### Memory Usage
free -h

### Disk Usage
df -h

### Running Processes
ps -ef

## 📊 Outcome
- Learned how to monitor CPU, memory, and disk
- Understood system performance basics
- Practiced real-time troubleshooting

## 🚀 Future Improvements
- Add shell scripts for automation
- Integrate monitoring tools like Nagios/Splunk
